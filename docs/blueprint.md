# **App Name**: CryptoVerse Explorer

## Core Features:

- Homepage Blog Feed: Display a paginated list of blog posts with featured images on the homepage (5 pages).
- Blog Post Detail View: Dedicated page for each blog post, accessible by clicking on the image or title from the homepage.  Includes content with formatted text, images, and links.
- Image Click-Through: Enable users to click on any image within a blog post, opening the image in a new tab.
- Related Blog Posts: Display a 'Related Posts' section (4-5 posts) based on tags and category for each detailed blog view.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and security.
- Background color: Light gray (#F5F5F5) for a clean and readable layout.
- Accent color: Vibrant orange (#FF5722) for calls to action and highlights.
- Body text: 'PT Sans', a modern sans-serif font for readability.
- Headline text: 'Space Grotesk', a technological sans-serif font to make section headlines pop; for longer text, use PT Sans for body
- Use a set of modern, minimalist icons to represent different categories or actions (e.g., sharing, liking, commenting).
- A clean, responsive layout with a clear hierarchy for easy navigation and readability.
